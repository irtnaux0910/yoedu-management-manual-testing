Manual Testing assignment for the YoEdu Management System - Classrooms and Students modules.
# MANUAL TESTING ASSIGNMENT - YOEDU MANAGEMENT SYSTEM

## Scope of Testing
This project focuses on executing Manual Testing in a Web environment for the **YoEdu Management System** (Website: https://management.yoedu.vn/).

The testing scope covers two main assigned modules:
1. **Classrooms Module (Phòng học):** Designing and executing test cases for classroom management features (View list, Add new, Edit, Delete, Search/Filter, etc.).
2. **Students Module (Học viên):** Designing and executing test cases for student information management (Register new student, Update information, Search, Status management, etc.).

## Repository Structure
This repository contains the following deliverables:
-  **`Test_Cases.xlsx`**: An Excel file containing all designed test cases for both modules, covering both positive and negative testing scenarios.
-  **`Bug_Reports.xlsx`**: An Excel file summarizing and reporting all bugs discovered during the testing execution, including steps to reproduce, severity levels, and attached evidence (images/videos).

##  Testing Environment
- **Operating System:** Windows 11 / macOS
- **Web Browser:** Google Chrome (Latest Version)
- **Documentation Tool:** Microsoft Excel
